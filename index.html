Fonctionnalités

  • Bloquez les contenus indésirables : Cache automatiquement les tweets provenant d'une liste d'utilisateurs promotionnels adultes.
  • Messages personnalisables : La version gratuite remplace les tweets bloqués par des messages informatifs, que nous mettons à jour régulièrement.
  • Intégration transparente : Fonctionne en arrière-plan sans affecter la vitesse de navigation.
  • Communauté participative : Notre base de données d'utilisateurs est continuellement mise à jour grâce aux contributions des utilisateurs.

Instructions d'installation

Comment installer CleanTwitter

Pour Google Chrome :

  1. Téléchargez les fichiers de l'extension :
    • Télécharger CleanTwitter Version Gratuite
    • Télécharger CleanTwitter Version Premium
  2. Activez le mode développeur :
    • Ouvrez Chrome et naviguez vers chrome://extensions/.
    • Activez le bouton Mode développeur en haut à droite.
  3. Chargez l'extension :
    • Cliquez sur Charger l'extension non empaquetée.
    • Sélectionnez le dossier contenant les fichiers de l'extension CleanTwitter.
  4. Actualisez Twitter :
    • Allez sur Twitter et actualisez la page pour activer CleanTwitter.

Pour Mozilla Firefox :

  1. Téléchargez les fichiers de l'extension :
    • Télécharger CleanTwitter Version Gratuite
    • Télécharger CleanTwitter Version Premium
  2. Chargez l'extension temporairement :
    • Ouvrez Firefox et naviguez vers about:debugging#/runtime/this-firefox.
    • Cliquez sur Charger un module complémentaire temporaire.
    • Sélectionnez le fichier .xpi de CleanTwitter.
  3. Actualisez Twitter :
    • Allez sur Twitter et actualisez la page pour activer CleanTwitter.

Politique de confidentialité

Votre vie privée compte pour nous

Dernière mise à jour : [Date]

Chez CleanTwitter, nous nous engageons à protéger votre vie privée. Cette politique explique comment nous traitons vos données lorsque vous utilisez notre extension.

Informations que nous collectons :

  • Stockage de données local : L'extension stocke certaines données localement sur votre appareil, telles que :
    • La liste des noms d'utilisateurs à bloquer.
    • Paramètres et préférences personnalisés.

Ce que nous ne collectons pas :

  • Données personnelles : Nous ne collectons ni ne transmettons aucune information personnelle.
  • Historique de navigation : Nous n'accédons pas à votre historique de navigation et ne le stockons pas.
  • Contenu utilisateur : Nous ne lisons ni ne stockons aucun contenu de votre fil Twitter.

Comment nous utilisons les informations :

Toutes les données stockées localement sont utilisées uniquement pour fournir les fonctionnalités de l'extension, comme le blocage de contenus spécifiques.

Services tiers :

Nous ne partageons vos données avec aucun service tiers.

Vos choix :

  • Contrôle : Vous pouvez désactiver ou désinstaller l'extension à tout moment via les paramètres de votre navigateur.
  • Transparence : Notre code est disponible pour examen sur GitHub.

Contactez-nous :

Si vous avez des questions ou des préoccupations concernant cette politique de confidentialité, veuillez nous contacter à your-email@example.com.
